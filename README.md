# TrollApps
一款专为巨魔TrollStore应用的资源集合类App
技术qq群：619802082 开发者十三哥QQ:350722326 WeChat:shisange2026
# TrollApps 功能
1.支持用户上传ipa zip dylib 等资源类文件</br>
2.友好的展示列表(社交能力)支持评论点赞分享等功能</br>
3.支持社交能力 用户中心 用户粉丝 聊天 用户分享</br>
4.支持版本管理 在线更新</br>
5.支持识别不同文件类型的导入和安装</br>
6.支持下载管理 签名管理</br>
7.支持创建WEB页面能力 分享有趣工具</br>
8.支持自定义铃声设置</br>
9. 。。。。。</br>
